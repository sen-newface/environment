docker環境にアクセスするために、 `/etc/hosts` に下記情報を記入してください

```
#sen-newface
127.0.0.1 localapp.jp
127.0.0.1 ethnam.localapp.jp
127.0.0.1 laravel.localapp.jp
127.0.0.1 yeahcheese.localapp.jp
127.0.0.1 plane.localapp.jp
```
